# Supporting research (historical)

These files are the raw output of the audit process that produced [`../crate-consolidation-plan.md`](../crate-consolidation-plan.md). They are **not** canonical and are **not** kept in sync with the plan. They exist as evidence of the adversarial-review process.

| File                   | What it is                                                                                                                                                                                                                                                                                                                                                       | Status                              |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------- |
| `research-regions.md`  | Per-region crate inventory from 12 parallel explorers. Each region was mapped independently, so counts/classifications differ slightly between regions.                                                                                                                                                                                                          | Superseded by §1/§2 of the plan.    |
| `research-catalogs.md` | Workaround catalogs (`extern "Rust"`, `dyn`/vtable, PORTING.md todos, tiny-crate classification, build-impact) from 5 parallel catalogers.                                                                                                                                                                                                                       | Superseded by §3/§5/§6 of the plan. |
| `initial-proposal.md`  | The **first-pass** synthesis, before adversarial review. Truncated mid-sentence (output cap). Contains known errors: `dns` in `bun_crypto`, `PnpmMatcher` moved to `bun_install`, `bun_dispatch` "deleted outright", `bun_jsc` dep list too narrow, event-loop design using `dyn` trait pointers. Every one of these was caught by review and fixed in the plan. | Superseded by the plan; do not use. |
| `research-reviews.md`  | The 24 adversarial reviews that found those errors. Different reviewers reached different LOC-deletion estimates (5-8k vs 12-15k) because they counted different categories; the plan's §6 is the reconciled table.                                                                                                                                              | Superseded by §7 of the plan.       |

Where these files disagree with [`../crate-consolidation-plan.md`](../crate-consolidation-plan.md), the plan is authoritative.
