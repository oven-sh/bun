---
title: Convert an absolute path to a file URL
sidebarTitle: Path to file URL
mode: center
---

Use `Bun.pathToFileURL()` to convert an absolute path to a `file://` URL.

```ts
Bun.pathToFileURL("/path/to/file.txt");
// => "file:///path/to/file.txt"
```
