Bun aims for complete Node.js API compatibility. Most `npm` packages intended for `Node.js` environments will work with Bun out of the box; the best way to know for certain is to try it.

This page is updated regularly to reflect compatibility status of the latest version of Bun. If you run into any bugs with a particular package, please [open an issue](https://bun.sh/issues). Opening issues for compatibility bugs helps us prioritize what to work on next.

## Built-in modules

### [`node:assert`](https://nodejs.org/api/assert.html)

游릭 Fully implemented.

### [`node:async_hooks`](https://nodejs.org/api/async_hooks.html)

游리 Only `AsyncLocalStorage`, and `AsyncResource` are implemented.

### [`node:buffer`](https://nodejs.org/api/buffer.html)

游릭 Fully implemented.

### [`node:child_process`](https://nodejs.org/api/child_process.html)

游리 Missing `Stream` stdio, `proc.gid`, `proc.uid`. IPC has partial support and only current only works with other `bun` processes.

### [`node:cluster`](https://nodejs.org/api/cluster.html)

游댮 Not implemented.

### [`node:console`](https://nodejs.org/api/console.html)

游리 Missing `Console` constructor.

### [`node:crypto`](https://nodejs.org/api/crypto.html)

游리 Missing `crypto.Certificate` `crypto.ECDH` `crypto.KeyObject` `crypto.X509Certificate` `crypto.checkPrime{Sync}` `crypto.createPrivateKey` `crypto.createPublicKey` `crypto.createSecretKey` `crypto.diffieHellman` `crypto.generateKey{Sync}` `crypto.generateKeyPair{Sync}` `crypto.generatePrime{Sync}` `crypto.getCipherInfo` `crypto.{get|set}Fips` `crypto.hkdf` `crypto.hkdfSync` `crypto.secureHeapUsed` `crypto.setEngine` `crypto.sign` `crypto.verify`. Some methods are not optimized yet.

### [`node:dgram`](https://nodejs.org/api/dgram.html)

游댮 Not implemented.

### [`node:diagnostics_channel`](https://nodejs.org/api/diagnostics_channel.html)

游릭 Fully implemented.

### [`node:dns`](https://nodejs.org/api/dns.html)

游릭 Fully implemented.

### [`node:domain`](https://nodejs.org/api/domain.html)

游릭 Fully implemented.

### [`node:events`](https://nodejs.org/api/events.html)

游리 Missing `on`.

### [`node:fs`](https://nodejs.org/api/fs.html)

游리 Missing `fs.fdatasync{Sync}` `fs.opendir{Sync}`. `fs.promises.open` incorrectly returns a file descriptor instead of a `FileHandle`.

### [`node:http`](https://nodejs.org/api/http.html)

游릭 Fully implemented.

### [`node:http2`](https://nodejs.org/api/http2.html)

游댮 Not implemented.

### [`node:https`](https://nodejs.org/api/https.html)

游릭 Fully implemented.

游댮 Not implemented.

### [`node:module`](https://nodejs.org/api/module.html)

游릭 Fully implemented.

### [`node:net`](https://nodejs.org/api/net.html)

游리 Missing `net.{get|set}DefaultAutoSelectFamily` `net.SocketAddress` `net.BlockList`.

### [`node:os`](https://nodejs.org/api/os.html)

游릭 Fully implemented.

### [`node:path`](https://nodejs.org/api/path.html)

游릭 Fully implemented.

### [`node:perf_hooks`](https://nodejs.org/api/perf_hooks.html)

游리 Only `perf_hooks.performance.now()` and `perf_hooks.performance.timeOrigin` are implemented. Recommended to use `performance` global instead of `perf_hooks.performance`.

### [`node:process`](https://nodejs.org/api/process.html)

游리 See `Globals > process`.

### [`node:punycode`](https://nodejs.org/api/punycode.html)

游릭 Fully implemented. _Deprecated by Node.js._

### [`node:querystring`](https://nodejs.org/api/querystring.html)

游릭 Fully implemented.

### [`node:readline`](https://nodejs.org/api/readline.html)

游릭 Fully implemented.

### [`node:repl`](https://nodejs.org/api/repl.html)

游댮 Not implemented.

### [`node:stream`](https://nodejs.org/api/stream.html)

游릭 Fully implemented.

### [`node:string_decoder`](https://nodejs.org/api/string_decoder.html)

游릭 Fully implemented.

### [`node:sys`](https://nodejs.org/api/util.html)

游리 See `node:util`.

### [`node:timers`](https://nodejs.org/api/timers.html)

游릭 Recommended to use global `setTimeout`, et. al. instead.

### [`node:tls`](https://nodejs.org/api/tls.html)

游리 Missing `tls.createSecurePair`.

### [`node:trace_events`](https://nodejs.org/api/tracing.html)

游댮 Not implemented.

### [`node:tty`](https://nodejs.org/api/tty.html)

游릭 Fully implemented.

### [`node:url`](https://nodejs.org/api/url.html)

游리 Missing `url.domainTo{ASCII|Unicode}`. Recommended to use `URL` and `URLSearchParams` globals instead.

### [`node:util`](https://nodejs.org/api/util.html)

游리 Missing `util.MIMEParams` `util.MIMEType` `util.getSystemErrorMap()` `util.getSystemErrorName()` `util.parseArgs()` `util.stripVTControlCharacters()` `util.transferableAbortController()` `util.transferableAbortSignal()`.

### [`node:v8`](https://nodejs.org/api/v8.html)

游댮 `serialize` and `deserialize` use JavaScriptCore's wire format instead of V8's. Otherwise, not implemented. For profiling, use [`bun:jsc`](/docs/project/benchmarking#bunjsc) instead.

### [`node:vm`](https://nodejs.org/api/vm.html)

游리 Core functionality works, but VM modules are not implemented. `ShadowRealm` can be used.

### [`node:wasi`](https://nodejs.org/api/wasi.html)

游리 Partially implemented.

### [`node:worker_threads`](https://nodejs.org/api/worker_threads.html)

游리 `Worker` doesn't support the following options: `eval`, `argv`, `execArgv`, `stdin`, `stdout`, `stderr`, `trackedUnmanagedFds`, `resourceLimits`. Missing `markAsUntransferable`, `moveMessagePortToContext`, `getHeapSnapshot`.

### [`node:zlib`](https://nodejs.org/api/zlib.html)

游리 Missing `zlib.brotli*`. Has not been optimized.
