<h1 align="center">Bun 文档</h1>

Bun官方文档：快速、一体化的JavaScript运行时。


## 发展

安装 [Mintlify CLI](https://www.npmjs.com/package/mint) 以在本地预览文档：

```bash
bun install -g mint
```

运行开发服务器：

```bash
mint dev
```

该网站将开放于 `http://localhost:3000`.
