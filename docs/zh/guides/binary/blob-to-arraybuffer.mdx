title: 将 Blob 转换为 ArrayBuffer
sidebarTitle: "Blob 到 ArrayBuffer"
mode: center
---

[Blob](https://developer.mozilla.org/zh-CN/docs/Web/API/Blob) 类提供了多种方法以不同格式消费其内容，包括 `.arrayBuffer()`。

```ts
const blob = new Blob(["hello world"]);
const buf = await blob.arrayBuffer();
```
