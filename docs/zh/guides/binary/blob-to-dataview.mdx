title: 将 Blob 转换为 DataView
sidebarTitle: "Blob 到 DataView"
mode: center
---

[Blob](https://developer.mozilla.org/zh-CN/docs/Web/API/Blob) 类提供了多种方法以不同格式消费其内容。此代码片段将内容读取到 [ArrayBuffer](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/ArrayBuffer) 中，然后从该缓冲区创建 [DataView](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/DataView)。

```ts
const blob = new Blob(["hello world"]);
const arr = new DataView(await blob.arrayBuffer());
```
