---
title: 将 DataView 转换为字符串
sidebarTitle: "DataView 到 字符串"
mode: center
---

如果 [DataView](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/DataView) 包含 ASCII 编码的文本，您可以使用 [TextDecoder](https://developer.mozilla.org/zh-CN/docs/Web/API/TextDecoder) 类将其转换为字符串。

```ts
const dv: DataView = ...;
const decoder = new TextDecoder();
const str = decoder.decode(dv);
```
