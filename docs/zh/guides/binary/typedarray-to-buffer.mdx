---
title: 将 Uint8Array 转换为 Buffer
sidebarTitle: "Uint8Array 到 Buffer"
mode: center
---

[Buffer](https://nodejs.org/api/buffer.html) 类扩展了 [Uint8Array](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Uint8Array)，添加了许多额外的方法。使用 `Buffer.from()` 从 [Uint8Array](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Uint8Array) 创建 [Buffer](https://nodejs.org/api/buffer.html) 实例。

```ts
const arr: Uint8Array = ...
const buf = Buffer.from(arr);
```
