这将启动一个在端口 `3000` 上监听的 HTTP 服务器。它对所有请求都返回一个状态为 `200`、内容为 "欢迎来到 Bun!" 的 `Response`。

详情请参见 [`Bun.serve`](/runtime/http/server)。

```ts server.ts icon="/icons/typescript.svg"
const server = Bun.serve({
  port: 3000,
  fetch(request) {
    return new Response("欢迎来到 Bun!");
  },
});

console.log(`监听 ${server.url}`);
```
