要将 npm 包添加为依赖，请使用 `bun add`。

```sh terminal icon="terminal"
bun add zod
```

---

这会将包添加到 [package.json] 中的 `dependencies`。默认情况下，将使用 `^` 范围指定符，表示任何未来的小版本或补丁版本都是可接受的。

```json package.json icon="file-json"
{
  "dependencies": {
    "zod": "^3.0.0" // [!code ++]
  }
}
```

---

要"固定"到包的确切版本，请使用 `--exact`。这会将包添加到 `dependencies` 而不使用 `^`，将您的项目固定到您安装的确切版本。

```sh terminal icon="terminal"
bun add zod --exact
```

---

要指定确切版本或标签：

```sh terminal icon="terminal"
bun add zod@3.0.0
bun add zod@next
```

---
