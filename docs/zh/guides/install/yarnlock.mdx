<Note>
  Bun v1.1.39 引入了 [bun.lock]，一种 JSONC 格式的锁定文件。[bun.lock] 是人类可读的，并且无需配置即可进行 git 对比，且不影响性能。在 1.2.0+ 版本中，它是新项目使用的默认格式。[**了解更多。**](/pm/lockfile#text-based-lockfile)
</Note>

---

使用 `--yarn` 标志生成与 Yarn 兼容的 [yarn.lock] 文件（除了 [bun.lockb] 之外）。

```sh terminal icon="terminal"
bun install --yarn
```

---

要将此设置为默认行为，请将以下内容添加到您的 [bunfig.toml] 文件中。

```toml bunfig.toml icon="settings"
[install.lockfile]
print = "yarn"
```

---

要在不写入磁盘的情况下将 Yarn 锁定文件打印到控制台，请使用 `bun` "运行" 您的 [bun.lockb]。

```sh terminal icon="terminal"
bun bun.lockb
```
