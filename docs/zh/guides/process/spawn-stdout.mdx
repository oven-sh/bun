---
title: 从子进程中读取 stdout
sidebarTitle: 读取 stdout
mode: center
---

当使用 [`Bun.spawn()`](/runtime/child-process) 时，子进程的 `stdout` 可以通过 `proc.stdout` 作为 `ReadableStream` 来消费。

```ts
const proc = Bun.spawn(["echo", "hello"]);

const output = await proc.stdout.text();
output; // => "hello"
```

---

要将子进程的 `stdout` 传输到父进程的 `stdout`，请设置为 "inherit"。

```ts
const proc = Bun.spawn(["echo", "hello"], {
  stdout: "inherit",
});
```
