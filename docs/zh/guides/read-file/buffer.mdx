sidebarTitle: 读取为 Buffer

---

`Bun.file()` 函数接受一个路径并返回一个 [BunFile] 实例。[BunFile] 类扩展了 `Blob` 并允许您以多种格式延迟读取文件。

要将文件读取到 `Buffer` 实例中，首先使用 `.arrayBuffer()` 将文件作为 `ArrayBuffer` 消费，然后使用 `Buffer.from()` 从 `ArrayBuffer` 创建一个 `Buffer`。

```ts index.ts icon="/icons/typescript.svg"
const path = "/path/to/package.json";
const file = Bun.file(path);

const arrbuf = await file.arrayBuffer();
const buffer = Buffer.from(arrbuf);
```

---

有关在 Bun 中使用 `Buffer` 和其他二进制数据格式的更多信息，请参阅 [二进制数据 > Buffer](/runtime/binary-data#buffer)。
