---
title: 获取文件的 MIME 类型
sidebarTitle: 获取 MIME 类型
mode: center
---

`Bun.file()` 函数接受一个路径并返回一个 [BunFile] 实例。[BunFile] 类扩展了 `Blob`，所以使用 `.type` 属性来读取 MIME 类型。

```ts
const file = Bun.file("./package.json");
file.type; // application/json

const file = Bun.file("./index.html");
file.type; // text/html

const file = Bun.file("./image.png");
file.type; // image/png
```

---

有关使用 [BunFile] 的更多信息，请参阅 [API > 文件 I/O](/runtime/file-io)。
