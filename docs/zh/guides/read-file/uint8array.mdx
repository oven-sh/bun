---
title: 将文件读取为 Uint8Array
sidebarTitle: 读取为 Uint8Array

---

`Bun.file()` 函数接受一个路径并返回一个 [BunFile] 实例。[BunFile] 类扩展了 `Blob` 并允许您以多种格式延迟读取文件。

要将文件读取到 `Uint8Array` 实例中，请使用 `.bytes()` 获取 [BunFile] 的内容。

```ts
const path = "/path/to/package.json";
const file = Bun.file(path);

const byteArray = await file.bytes();

byteArray[0]; // 第一个字节数组
byteArray.length; // 字节数组长度
```

---

有关在 Bun 中使用 `Uint8Array` 和其他二进制数据格式的更多信息，请参阅 [API > 二进制数据 > 类型化数组](/runtime/binary-data#typedarray)。
