要删除文件，请使用 `Bun.file(path).delete()`。

```ts delete-file.ts icon="/icons/typescript.svg"
// 删除一个文件
const file = Bun.file("path/to/file.txt");
await file.delete();

// 现在这个文件不存在了
const exists = await file.exists();
// => false
```
