sidebarTitle: 导入 JSON

---

Bun 原生支持 `.json` 导入。

```json package.json icon="file-json"
{
  "name": "bun",
  "version": "1.0.0",
  "author": {
    "name": "John Dough",
    "email": "john@dough.com"
  }
}
```

---

像任何其他源文件一样导入该文件。

```ts data.ts icon="/icons/typescript.svg"
import data from "./package.json";

data.name; // => "bun"
data.version; // => "1.0.0"
data.author.name; // => "John Dough"
```

---

Bun 还支持 [导入属性](https://github.com/tc39/proposal-import-attributes/) 和 [JSON 模块](https://github.com/tc39/proposal-json-modules) 语法。

```ts data.ts icon="/icons/typescript.svg"
import data from "./package.json" with { type: "json" };

data.name; // => "bun"
data.version; // => "1.0.0"
data.author.name; // => "John Dough"
```

---

有关在 Bun 中使用 TypeScript 的更多信息，请参阅 [文档 > 运行时 > TypeScript](/runtime/typescript)。
