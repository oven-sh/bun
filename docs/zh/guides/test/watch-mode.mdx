使用 `--watch` 标志以监视模式运行测试。

```sh terminal icon="terminal"
bun test --watch
```

---

这将在检测到文件更改时重启运行中的 Bun 进程。它很快。在此示例中，编辑器已配置为每次按键时保存文件。
