`Bun.escapeHTML()` 实用工具可用于转义字符串中的 HTML 字符。将进行以下替换。

- `"` 变为 `"&quot;"`
- `&` 变为 `"&amp;"`
- `'` 变为 `"&#x27;"`
- `<` 变为 `"&lt;"`
- `>` 变为 `"&gt;"`

此函数针对大输入进行了优化。非字符串类型将在转义前转换为字符串。

```ts
Bun.escapeHTML("<script>alert('Hello World!')</script>");
// &lt;script&gt;alert(&#x27;Hello World!&#x27;)&lt;&#x2F;script&gt;
```
