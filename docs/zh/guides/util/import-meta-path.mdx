---
title: 获取当前文件的绝对路径
sidebarTitle: import.meta.path
mode: center
---

Bun 在 [`import.meta`](/runtime/module-resolution#import-meta) 对象上提供了一些模块特定的实用工具。使用 `import.meta.path` 检索当前文件的绝对路径。

```ts /a/b/c.ts icon="/icons/typescript.svg"
import.meta.path; // => "/a/b/c.ts"
```

---

请参阅 [文档 > API > import.meta](/runtime/module-resolution#import-meta) 获取完整文档。
