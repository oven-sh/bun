`Bun.main` 属性包含当前入口点的绝对路径。

<CodeGroup>

```ts foo.ts icon="/icons/typescript.svg"
console.log(Bun.main);
```

```ts index.ts icon="/icons/typescript.svg"
import "./foo.ts";
```

</CodeGroup>

---

打印的路径对应于使用 `bun run` 执行的文件。

```sh terminal icon="terminal"
bun run index.ts
```

```txt
/path/to/index.ts
```

```sh terminal icon="terminal"
bun run foo.ts
```

```txt
/path/to/foo.ts
```
