以语义化版本格式获取当前 Bun 版本。

```ts index.ts icon="/icons/typescript.svg"
Bun.version; // => "1.3.3"
```

---

获取编译此 Bun 二进制文件所用的 [`oven-sh/bun`](https://github.com/oven-sh/bun) 的确切 `git` 提交。

```ts index.ts icon="/icons/typescript.svg"
Bun.revision; // => "49231b2cb9aa48497ab966fc0bb6b742dacc4994"
```
