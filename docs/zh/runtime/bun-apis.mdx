---

Bun 在 `Bun` 全局对象和一系列内置模块上实现了一组原生 API。这些 API 经过高度优化，代表了实现某些常见功能的规范"Bun 原生"方式。

Bun 努力在可能的地方实现标准 Web API。Bun 主要为服务器端任务引入新 API，这些任务没有现有标准，例如文件 I/O 和启动 HTTP 服务器。在这些情况下，Bun 的方法仍然是基于标准 API 构建的，如 `Blob`、`URL` 和 `Request`。

```ts server.ts icon="/icons/typescript.svg"
Bun.serve({
  fetch(req: Request) {
    return new Response("Success!");
  },
});
```
