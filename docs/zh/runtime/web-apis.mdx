---
title: "Web APIs"
description: "Bun 支持的用于服务端 JavaScript 的 Web 标准 API"
mode: center
---

一些 Web API 在像 Bun 这样的以服务器优先的运行时环境中并不相关，例如 [DOM API](https://developer.mozilla.org/en-US/docs/Web/API/HTML_DOM_API#html_dom_api_interfaces) 或 [History API](https://developer.mozilla.org/en-US/docs/Web/API/History_API)。然而，许多其他 API 在浏览器上下文之外也非常有用；只要可能，Bun 实现这些 Web 标准 API 而不是引入新 API。

以下 Web API 部分或完全支持。
